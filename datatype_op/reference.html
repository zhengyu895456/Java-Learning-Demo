引用数据类型

在 Java 语言中，除了8种基本数据类型外，其他的都是引用类型，指向各种不同的对象（类类型、接口类型、数组类型），理解了引用可以更好的帮助掌握 Java 对象生命周期和 JVM 内部机制。

Java 引用类型包含以下四类，详细的内容可以在 JVM 章节时去学习，这里只提前列举一下。

  • 软引用
  • 弱引用
