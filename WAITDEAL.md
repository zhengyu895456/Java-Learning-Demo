# 数据类型

Java 是一种强类型语言，我们必须为每一个变量声明一种类型。数据类型限制了变量可以持有的值和表达式可以产生的值，以及在这些值上可以进行的操作。强类型机制有助于在编译时探测错误。

在 Java 中，数据类型分为两类：基本数据类型（Primitive Type）和引用数据类型（Reference Type）。
